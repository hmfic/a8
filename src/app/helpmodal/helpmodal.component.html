About the Application8 demo

  • Application8 mocks a simple social media system using pubically available dummy data
  • Everything is my original work (no borrowing from other places)
  • Everything was done with Angular 6 (6.1.6), Typescript (2.7.2) and NodeJS (10.9.0)
  • Everything uses Material Design for UI components
  • Charts are dynamically created using hand-built D3.js(v4)
  • The dummy data is available at https://jsonplaceholder.typicode.com/
  • The data is dynamically retrieved in JSON using asynchronous REST calls
  • The geolocation service is dynamically retrieved from http://ip-api.com/json
  • Uses SCSS and Material Design color palettes
  • Maps are built using Google Maps
  • Settings are stored in the browser local store so they persist across sessions
  • Everything is stored and managed in git https://github.com/hmfic/a8
  • Responsive - runs on iPads, etc open_in_new
  • It took around 1700 lines of Typescript and 1.5 weeks elapsed time (in the evenings and weekend)
